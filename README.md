# 📐 AI Math Teacher - Pakistani Style Chatbot

A smart, interactive web-based chatbot built using Python, Streamlit, and the Groq Cloud API (Llama-3.3-70b). The chatbot is system-prompted to behave like a Pakistani Math Teacher who explains complex mathematical concepts and logic in English with a local touch.

## 🚀 Features

* **Custom Persona:** Acts specifically as a Pakistani Math Teacher.
* **Real-time Chat Interface:** Built using Streamlit's smooth `st.chat_input` and chat elements.
* **Ultra-Fast Responses:** Powered by Groq Cloud's high-speed `llama-3.3-70b-versatile` model.
* **Session State Management:** Keeps track of the conversation history during the session.

## 🛠️ Tech Stack & Prerequisites

* **Frontend:** Streamlit
* **LLM Provider:** Groq Cloud API (Llama 3.3)
* **Language:** Python 3.x
* **Libraries:** `streamlit`, `groq`, `python-dotenv`
